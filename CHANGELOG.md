## [0.1.0] - 2025-02-06

### Added

- Initial release of BlazeQC
- Fast quality control for sequencing data in Mojo
- Stats modules: adapter content, base composition, duplication, k-mer content, length/quality distribution, and related QC metrics
- HTML report generation
- Conda recipe and pixi-build (rattler-build) support for packaging
